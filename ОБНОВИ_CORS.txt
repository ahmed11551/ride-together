==========================================
ОБНОВЛЕНИЕ CORS ПОСЛЕ ДЕПЛОЯ VERCEL
==========================================

ШАГ 1: ПОЛУЧИТЕ ДОМЕН VERCEL

После деплоя на Vercel вы получите домен вида:
https://your-app-abc123.vercel.app

Запишите его!

ШАГ 2: ОБНОВИТЕ ALLOWED_ORIGINS В TIMEWEB

1. Откройте Timeweb Cloud → Ваше бэкенд приложение
2. Перейдите в Environment Variables
3. Найдите ALLOWED_ORIGINS
4. Обновите значение на:

https://your-app-abc123.vercel.app,https://*.vercel.app

(Замените your-app-abc123.vercel.app на ваш реальный домен)

ШАГ 3: ПЕРЕЗАПУСТИТЕ БЭКЕНД

В Timeweb Cloud → Ваше приложение → Перезапустить

ШАГ 4: ПРОВЕРКА

Откройте ваш Vercel домен и проверьте:
- Нет ошибок CORS в консоли (F12)
- Регистрация/вход работают
- Запросы к API успешны
