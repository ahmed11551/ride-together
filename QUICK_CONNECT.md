# ⚡ Быстрое подключение к серверу

## Самый простой способ:

Откройте терминал и выполните:

```bash
ssh root@194.67.124.123
```

Введите пароль от сервера (пароль не отображается при вводе).

---

## Или используйте скрипт:

```bash
cd /Users/ahmeddevops/Desktop/ride/ride-together
./connect.sh
```

---

## После подключения:

```bash
# Перейти в директорию проекта
cd /var/www/ride-together/server

# Запустить установку улучшений
./INSTALL_IMPROVEMENTS.sh
```

---

## Если не знаете пароль:

1. Зайдите на https://www.reg.ru
2. Войдите в личный кабинет
3. Найдите ваш VPS сервер
4. Используйте веб-консоль или сбросьте пароль

---

## Проблемы?

- **"Connection refused"** → Сервер выключен, включите через панель REG.RU
- **"Permission denied"** → Неверный пароль, сбросьте через панель REG.RU
- **"Host key verification failed"** → Выполните: `ssh-keygen -R 194.67.124.123`
