==========================================
НАСТРОЙКА ФРОНТЕНДА НА VERCEL
==========================================

ШАГ 1: СОЗДАНИЕ ПРОЕКТА

1. Откройте https://vercel.com/dashboard
2. Add New Project
3. Выберите ваш GitHub репозиторий
4. Framework Preset: Vite (или оставьте пустым)
5. Root Directory: ride-together (или оставьте пустым)
6. Build Command: npm run build (или оставьте пустым)
7. Output Directory: dist (или оставьте пустым)

ШАГ 2: ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ

В разделе Environment Variables добавьте:

VITE_API_URL=https://ваш-бэкенд-url.twc1.net

(Замените на реальный URL вашего бэкенда из Timeweb)

Опционально:
VITE_WS_URL=wss://ваш-бэкенд-url.twc1.net
VITE_YANDEX_MAPS_API_KEY=ваш-ключ
VITE_TELEGRAM_BOT_TOKEN=ваш-токен
VITE_SENTRY_DSN=ваш-sentry-dsn

ШАГ 3: ДЕПЛОЙ

Нажмите Deploy и дождитесь завершения

ШАГ 4: ПОСЛЕ ДЕПЛОЯ

Запишите ваш Vercel домен (например: your-app.vercel.app)
Он понадобится для обновления CORS на бэкенде
