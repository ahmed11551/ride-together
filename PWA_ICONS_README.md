# PWA Иконки

Для полноценной работы PWA нужны иконки приложения.

## Требуемые иконки

1. **icon-192.png** - 192x192 пикселей
2. **icon-512.png** - 512x512 пикселей

## Где разместить

Иконки должны быть размещены в папке `public/`:
- `public/icon-192.png`
- `public/icon-512.png`

## Как создать иконки

### Вариант 1: Онлайн генераторы
1. Используйте [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
2. Или [RealFaviconGenerator](https://realfavicongenerator.net/)

### Вариант 2: Вручную
1. Создайте логотип/иконку приложения
2. Экспортируйте в PNG формате:
   - 192x192 пикселей для `icon-192.png`
   - 512x512 пикселей для `icon-512.png`
3. Разместите в `public/`

### Вариант 3: Использовать существующий favicon
Если у вас есть favicon.ico, можно конвертировать его:
```bash
# Используя ImageMagick
convert favicon.ico -resize 192x192 icon-192.png
convert favicon.ico -resize 512x512 icon-512.png
```

## Рекомендации по дизайну

- Используйте простой, узнаваемый символ
- Избегайте мелких деталей
- Используйте контрастные цвета
- Убедитесь, что иконка хорошо выглядит на светлом и темном фоне
- Для maskable иконок оставьте безопасную зону (padding) вокруг основного контента
