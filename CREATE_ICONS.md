# Создание иконок для PWA

Иконки `icon-192.png` и `icon-512.png` отсутствуют. Нужно их создать.

## Вариант 1: Использовать онлайн генератор

1. Перейдите на https://favicon.io/favicon-generator/
2. Создайте иконки 192x192 и 512x512
3. Сохраните их как `icon-192.png` и `icon-512.png` в `public/`

## Вариант 2: Создать простые иконки из favicon.svg

Можно использовать ImageMagick или другой инструмент для конвертации SVG в PNG нужных размеров.

## Вариант 3: Временно использовать placeholder

Можно временно использовать favicon.ico для обеих иконок, но лучше создать правильные иконки.

После создания иконок:
1. Поместите их в `public/`
2. Пересоберите frontend: `npm run build`
3. Загрузите на сервер
